                                                  是一坨数字，根据题目名称想到塔珀自指公式（Tupper’s Self-Referential Formula）

                                                  12<⌊mod(⌊y17⌋2−17⌊x⌋−mod(⌊y⌋,17),2)⌋,(0≤x≤105,k≤y≤k+16)

                                                  根据k取值不同，这个方程可以呈现出不同的样子，当k取刚刚解码得到的一串数字时，我们用如下脚本画出图像：

                                                  import numpy as np
import matplotlib.pyplot as plt

def Tupper_self_referential_formula(k): 
    aa = np.zeros((17,106))
    def f(x, y):
        y += k
        a1 = 2**-(-17*x - y%17)
        a2 = (y // 17) // a1
        return 1 if a2 % 2 > 0.5 else 0
 
    for y in range(17):
        for x in range(106):
            aa[y, x] = f(x, y)
  
    return np.flip(aa[:,::-1], axis=1)

                                                  # PAD

                                                  这题我可能是用了revenge的方法，导致revenge被我秒了？？

                                                  我们浅浅地记输出的每行四个部分分别是(ni,ei,(epi,ci))，然后对于每一行的m都是一样的，所以考虑广播攻击的形式。每一行都符合：

                                                  ci≡(epi512+mepi)ei(modni)

                                                  每个方程都只有m是未知的，并且有40个方程，一元但是40个方程，简直是赤裸裸的挑衅……所以我们有理由只选取少数几个具有特殊性质的方程来帮我们找到突破点。

                                                  首先看ei作为二项式的指数非常不爽，筛选出ei=1的数据，剩下4个，这四个满足：

                                                  ci≡epi512+mepi(modni)

                                                  恰好还有两个人的epi=3，选择这两个人，它们符合：

                                                  ci≡3512+m3(modni)

                                                  两个式子构成同余式组：

                                                  {m3≡c1−3512(modn1)m3≡c2−3512(modn2)

                                                  用中国剩余定理可以求出m3(mod≈22048)的结果，因为flag即m只有512个比特位，所以m3<n1n2，可以直接开方?
